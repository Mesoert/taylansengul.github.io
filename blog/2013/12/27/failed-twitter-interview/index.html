A Python Solution of Failed Twitter Interview Problem With Sketches

Dec 27th, 2013 | Comments

This is my shot at the failed Twitter interview problem with a few extras using Python. The problem is this, given an array of wall heights, find the amount of water filled between these walls after a heavy rain. Suppose the given array is [2,5,1,2,3,4,7,7,6], then the corresponding picture is:

       WW 
       WWW
 W     WWW
 W     WWW
 W W   WWW
WW W W WWW
WWWWWWWWWW
2513121776

To get the picture, we define the following function

from __future__ import print_function
def draw_walls(walls):
    """Print an array of wall heights"""
    M = max(walls)  # no of rows
    N = len(walls)  # no of columns
    for i in range(M, 0, -1):
        for j in range(N):
            if walls[j] >= i:
                print('W', end='')
            else:
                print(' ', end='')
        print()
    for i in range(N):
        print(walls[i], end='')

and then call

draw_walls([2,5,1,2,3,4,7,7,6])

Anyway, a clever solution is also given in the blog page. Here in addition to finding the total number of filled spaces, the following code will also sketch the walls and the water filled in the walls.

For example, calling

walls = [2, 5, 1, 3, 1, 2, 1, 7, 7, 6]

print("%d filled spaces" % filled)

will give

       WW 
       WWW
 W-----WWW
 W-----WWW
 W-W---WWW
WW-W-W-WWW
WWWWWWWWWW
